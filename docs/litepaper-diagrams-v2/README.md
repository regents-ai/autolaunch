# Autolaunch Litepaper Diagrams V2

This second set revises the diagram logic and presentation style for the April 22, 2026 litepaper draft.

The main correction is in the flywheel figure:

- the launch is shown as a one-time event
- the ongoing loop begins after launch
- the post-launch loop is the income flywheel, not the launch itself
