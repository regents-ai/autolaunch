# Autolaunch Litepaper Diagrams

These six diagrams were created for the April 22, 2026 litepaper draft in a shared voxel-style visual language inspired by the Autolaunch logo.

This folder holds the original first-pass set.

## Suggested use

1. `autolaunch-flywheel-voxel.png`
   The overall operating loop from edge to launch, revenue, and renewed runway.
2. `continuous-clearing-auction-voxel.png`
   A reader-friendly explanation of budget, max price, and clearing over time.
3. `launch-capital-and-vesting-voxel.png`
   Launch-day splits for auction tokens, LP tokens, vested supply, treasury runway, and fees.
4. `recognized-revenue-lane-voxel.png`
   The rule that only Base USDC routed into the subject revenue lane counts as recognized revenue.
5. `subject-revsplit-voxel.png`
   The subject-level split from recognized revenue to Regents Labs, stakers, and treasury remainder.
6. `regent-staking-flow-voxel.png`
   The company-level staking flow for protocol revenue, claims, buybacks, and first-year emissions.

## Style notes

- Formal editorial diagrams, not playful illustrations
- Mostly blue with cream voxel forms and restrained teal or gold accents
- Light background so the figures can sit comfortably inside the litepaper
